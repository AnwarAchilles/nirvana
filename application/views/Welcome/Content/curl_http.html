# Installation How To Use

HTTP Request with CURL

HTTP request to server based on CURL PHP, already installed as a library in Nirvana and there are methods in CoreController to make requests in methods such as GET, POST, PUT, DELETE

About the plugin : https://github.com/philsturgeon/codeigniter-curl
