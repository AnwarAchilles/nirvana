# Configurations Layouting Multitple Layouting View Data

Template Engine with TWIG

Twig is the template engine of the php framework namely sympony, this tool is already installed and configured with nirvana, to use it just hook the Controller to CoreController and use the layout method which is integrated with twig.

official documentations here : https://twig.symfony.com/
Configurations
There are 2 types of configuration that are already available in the /application/config/ directory

1. Twig Based Configurations
files are in /application/config/twig.php
2. Core Layout Configurations
files are in /application/config/core.php
